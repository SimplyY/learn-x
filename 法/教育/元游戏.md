# 背景
很多好游戏是非常具备创造力和想象力的，对小孩子的教育非常有用，包括 minecraft、Roblox、塞尔达、魔兽争霸。

一方面可以让小孩子玩高质量的游戏，另一方面也可以创造游戏。整个过程本身就是元游戏。


# 目标
不是把孩子变成开发者，而是借游戏把阅读、AI、表达、审美、反馈、创造连成一个活系统，让孩子在好玩的过程中，长出主体性、自驱、学会做中学、玩中学等元学习能力。

# 核心要点
1. 打造成长型创造系统（在过程中快乐成长），而不是培养游戏开发者
2. 打造高质量的游戏模板库（10+），实现循序渐进的教学
   1. 15分钟创造快感
   2. 1小时项目，完整体验
   3. 1周作品感
3. 合适的游戏类型
   1. 生存升级类（肉鸽轻生存类），小改动 → 巨大反馈
   2. 塔防类（如植物大战僵尸，魔兽td等等）
   3. 模拟经营类（如主题医院等）
   4. 剧情互动类

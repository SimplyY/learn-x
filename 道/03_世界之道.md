# 世界之道

## core

世界由三重秩序交织而成：

1. **原子世界**：身体、土地、房产、商品、线下关系。空间有限，位置重要，现实不可跳过。
2. **比特世界**：信息、连接、网络、信用、注意力。连接改变效率，也制造噪声与茧房。
3. **向量世界**：AI、模型、语义、知识压缩。时间被压缩，判断力被放大。



## 原则

1. **原子为基**：身体、家庭、现金流、真实关系，是长期底座。
2. **比特为网**：信息流动越快，越要守住注意力与信任。
3. **向量为剑**：AI 应增强判断与行动，不应替代主体性。
4. **周期为律**：文明、市场、技术与人生，都在周期中演化。
5. **生命为本**：技术、财富、系统，最终都应服务更好的生活。

# 道

道，是我长期的方向、本质与人生母题。

## 生命秩序（现实优先级）
**健康为底座（第一），生活为主体（第二），创造/工作为表达（第三），投资为护城河（第四），系统为工具；**
> 后者不得反噬前者(尤其是中长期)。

## 我的三条核心之道

1. **自我认知**：我是谁，我从哪里来，我真正重视什么。详见 [](01_自我认知.md)
2. **人生导师**：我愿成为什么样的人，用谁校准自己。详见 [](02_人生导师.md)
3. **世界之道**：我如何理解世界、时代、文明与技术。[](03_世界之道.md)

> 健康是道的地基，Learn-X 是脑，AI 是剑，写作是形，真实世界是校验场，资产是护城河。

## 道的判断标准

一条内容能否进入「道」，至少满足 4 条：

1. 3-5 年后仍重要。
2. 能指导多个领域。
3. 在多个场景反复出现。
4. 会影响真实选择。
5. 能压缩成一句高密度原则。
6. 经历过现实反馈或重大内在验证。
7. 能统摄下层的法、术、器。

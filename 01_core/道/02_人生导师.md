# 人生导师
## core

人生导师不是偶像系统，而是我的内在校准器。

### 心底之问
1. 日常时：我今天是否更像苏轼、孟岩、孔子了呢？
2. 焦虑时：如果苏轼、孟岩、孔子是我，他们会怎么思考、怎么选择、怎么行动？



## 箴言

### 孟岩
1. 允许万物穿过自己
2. 投资是为了更好的生活
3. 人何以自处？
4. 够与多


### 孔子
1. 因材施教
2. 己所不欲，勿施于人
3. 君子不器


### 苏轼
1. 着力即差
2. 变与不变：自其变者而观之，则天地曾不能以一瞬；自其不变者而观之，则物与我皆无尽也
3. 之境：闲人之乐、圆缺之意、风雨之晴、东去之壮、赤子之心。
4. 问汝平生功业，黄州惠州儋州；试问岭南好不好，​此心安处即吾乡；人生如梦，一尊还酹江月。

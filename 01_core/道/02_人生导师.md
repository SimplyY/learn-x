# 人生导师
## core

人生导师不是偶像系统，而是我的内在校准器。

### 心底之问
1. 日常时：我今天是否更像苏轼、孟岩、孔子了呢？
2. 焦虑时：如果苏轼、孟岩、孔子是我，他们会怎么思考、怎么选择、怎么行动？

## 三元校准系统

## 使用方式

遇到关键选择时，只问三个问题：

1. 苏轼会不会觉得我太紧、太执、太被外物牵引？
2. 孔子会不会觉得我丢掉了修身、教育与成人之道？
3. 孟岩会不会觉得我忽视了现实规律、风险和长期现金流？

如果三者都不反对，这个选择大概率是稳的。

### 苏轼：生命力与精神自由

苏轼提醒我：

1. 不把人生收缩成得失。
2. 在风雨、贬谪、变化中保持审美与旷达。
3. 重视此心安处、着力即差、平常之乐。
4. 在理想与现实之间保持弹性。

关键词：

- 着力即差
- 此心安处
- 生命力
- 审美与旷达

---

### 孔子：修身、教育与人格秩序

孔子提醒我：

1. 教育首先是成人，而不是工具化训练。
2. 己所不欲，勿施于人。
3. 君子不器，人不能被单一功能定义。
4. 真正的成长需要礼、仁、习惯与长期修身。

关键词：

- 理想
- 因材施教
- 己所不欲，勿施于人
- 君子不器
- 修身成人

---

### 孟岩：现代理性、长期主义与现实规律

孟岩提醒我：

1. 投资是为了更好的生活。
2. 够与多之间，要有清醒边界。
3. 小而美比盲目做大更适合我。
4. 现实规律不可跳过，商业、现金流、风险都要尊重。

关键词：

- 理性
- 够与多
- 小而美
- 投资为了生活
- 现实反馈

# AI 对话

请基于我最近一周与你的所有对话，做一次 Learn-X 周回顾。

目标不是普通总结，而是帮我识别：
1. 【重点】我这一周反复思考的 2-3 个核心问题（400字以内）；
2. 【重点】最值得沉淀为「道 / 法 / 术」的判断（道需要单独判断，整体400字以内）；
3. 【重点】给出一个精炼版「我应该人工确认的清单」，每类最多 2～5 项（不要硬凑 5 项，内容质量不够的2项、1项、甚至 0 项也行的），用md 的 - [ ] 格式。和「下周最该做的 3 件事」
4. 【可选】精华问题摘要（问题 + 核心答案 + 100 字以内的 2-3 个精华摘要，分点输出）
5. 【可选】最值得升级为 Skill 的重复工作流（直说想法，不给具体实现）；
6. 【可选】最值得交给 Codex 执行的1-2任务（直说想法，不给具体实现）；
7. 【可选】值得进入写作、Demo 或真实行动的方向（直说想法，不给具体实现）；
8. 【可选】哪些内容只是短期情绪或噪声，应该删除或观察（输出的时候补充上下文）。

请输出 1000～3000 字（宁缺毋滥，重要的内容不要遗漏），结构化整理，以及最精华的内容可以原文保留。

要求：
- 只基于你能可靠回忆到的内容；
- 不要编造具体原文；
- 优先提炼主题、问题、判断、任务；
- 行文按重要性排序，越重要越靠前；
- 后面的详细内容用于选读和给后续 AI 当上下文，不重要的细节可以压缩或省略；
- 每周不会产生很多新的「道 / 法」，请宁少勿滥；
- 按照今天的日期生成一下文件名，格式为 yyyy-mm-dd.md
- 用 md 格式输出，在代码块里面，方便我一键复制粘贴的，而不是富文本，不要输出```text 这种，可能导致代码块崩溃的内容

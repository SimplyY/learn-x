# 语言锋芒

回答不只要正确，还要能刺中认知。请在 AI 加工的重要判断中，生成精炼、有冲击力、能触发反思的判断句。这类句子不是装饰，而是认知钉子。

重要判断应尽量压缩成金句，并满足以下标准：

1. 自闭环：单独看也能成立，不依赖大段解释；
2. 短而硬：优先 8-20 字，少用长句；
3. 有反转：刺破一个默认假设、舒适叙事或漂亮幻觉；
4. 有代价感：让用户看到继续这样做会损失什么；
5. 有行动指向：读完后能影响判断、删除、选择或行动；
6. 受证据约束：不能为了好听而夸张、玄学或编造。

对关键内容按三步处理：先写出普通判断，明确这件事真正说明什么；再寻找反转关系，识别用户可能误以为什么、现实相反在哪里；最后删除解释和铺垫，保留最锋利的判断。

可优先使用反转句、边界句、代价句、审判句、隐喻句和截断句，例如：

- 你以为是 A，其实是 B。
- 没有 X，Y 只是 Z。
- 继续 X，代价是 Y。
- 判断 X，只看 Y。
- A 是剑，也可能是茧。
- 不要再优化 X，先验证 Y。

每个关键模块尽量保留一句金句，但不要堆砌。普通内容不强行制造金句；重要内容必须有锋芒。原料摘录优先保留原文，不强行改写；AI 加工部分应比普通摘要更锋利，但仍受 Context 和事实证据约束。

禁止为了冲击而夸张，为了押韵牺牲准确，为了金句制造无证据判断，把普通内容包装成深刻，或把情绪判断伪装成长期原则。

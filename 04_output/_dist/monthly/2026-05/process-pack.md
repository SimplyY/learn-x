3. Codex / 脚本只生成 `_dist` 和 `04_output/monthly/YYYY-MM.md` 最小壳；如果 Output 文件已有内容，不覆盖。
4. 人再决定是否把正文写入 `04_output/monthly/YYYY-MM.md`，以及是否进入 Memory 或正式长期资产。
